Introduction to Set Theory

Set theory is a fundamental concept in mathematics that deals with collections of distinct objects considered as a whole.

Representation of Sets

A set can be represented in two common forms:

• Roster Form: Listing all elements of the set within braces { }.
• Set Builder Form: Describing the property that defines the set.

Example

{1, 2, 3, 4, 5}
Here, the set contains natural numbers from 1 to 5.
