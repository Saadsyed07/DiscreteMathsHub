Introduction to Set Theory

A set is a well-defined collection of distinct objects considered as a single entity. These objects are called elements or members of the set.

Example:
A = {1, 2, 3, 4, 5} ‚Äî first five natural numbers.
B = {Apple, Mango, Banana} ‚Äî fruits.
C = {x | x is a vowel in the English alphabet} ‚Üí {a, e, i, o, u}.

Real-Life Examples

Set theory appears everywhere around us. A few common examples:

  • The set of all students in your class.
  • The set of even numbers between 1 and 20.
  • The set of books available in the library.
  • The set of movies you have watched this year.

Representation of Sets

Sets can be represented in two common ways:

Roster Form: Elements are listed inside curly braces.
Example: A = {1, 2, 3, 4}

Set Builder Form: Describes the property of elements.
Example: B = {x | x is an even number less than 10}
