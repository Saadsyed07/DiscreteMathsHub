Venn Diagrams

A Venn Diagram visually shows relationships between sets using overlapping circles. Introduced by John Venn, it illustrates union, intersection, and difference of sets.

Example:
Let A = {1, 2, 3, 4} and B = {3, 4, 5, 6}.
• A ∪ B = {1, 2, 3, 4, 5, 6} (Union)
• A ∩ B = {3, 4} (Intersection)
• A − B = {1, 2} (Difference)
