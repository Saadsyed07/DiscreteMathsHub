Properties of Sets

Sets follow certain algebraic laws that help simplify expressions and verify relationships between them. Major properties include:

  • Commutative Laws: A ∪ B = B ∪ A, and A ∩ B = B ∩ A
  • Associative Laws: (A ∪ B) ∪ C = A ∪ (B ∪ C)
  • Distributive Laws: A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
  • Identity Laws: A ∪ ∅ = A and A ∩ U = A
  • Complement Laws: A ∪ A′ = U and A ∩ A′ = ∅
  • De Morgan’s Laws: (A ∪ B)' = A' ∩ B' and (A ∩ B)' = A' ∪ B'
Example:
Let A = {1, 2, 3}, B = {3, 4, 5}.
A ∪ B = {1,2,3,4,5} and B ∪ A = {1,2,3,4,5}. Hence, the Commutative Law of Union holds.
