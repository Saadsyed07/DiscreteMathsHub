Subsets & Power Sets

Enter elements of a set to generate all possible subsets. The collection of all subsets forms the Power Set. Duplicates will be removed automatically.

💡 Understanding Subsets

  • Subset (A ⊆ B): Every element of A is also in B
  • Proper Subset (A ⊂ B): A is a subset of B, but A ≠ B
  • Power Set P(A): The set of all subsets of A
  • Formula: If |A| = n, then |P(A)| = 2n
