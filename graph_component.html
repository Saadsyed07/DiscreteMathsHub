Connected Components in Graphs

In Graph Theory, a connected component is a maximal set of vertices in which each pair of vertices is connected by a path.

Connected Component: A subgraph in which any two vertices are connected by a path, and no vertex in the subgraph connects to vertices outside it.

ðŸ”¹ Key Concepts

  • Connected Graph: Every vertex is reachable from every other vertex through some path
  • Disconnected Graph: A graph with two or more separate components
  • Connected Component: An isolated group of connected nodes that cannot reach nodes in other components
  • Strongly Connected (Directed): Every vertex is reachable from every other vertex
  • Weakly Connected (Directed): Connected when direction is ignored

Connected components reveal isolated groups and connectivity patterns across many domains:

  • Social Networks: Identify communities and friend circles
  • Computer Networks: Detect network partitions and isolated subnets
  • Image Analysis: Segment different objects in an image
  • Transportation: Find disconnected road networks or airline routes
  • Biology: Analyze protein interaction networks
