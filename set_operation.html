Set Operations

Set operations let us combine and compare sets to form new sets. Below you will find definitions, examples, and an interactive simulator — fully styled like the Set Definition page.

Common Operations

Union (A ∪ B)
The union returns all elements in A or B (or both).
Example: A = {1,2,3}, B = {3,4,5} → A ∪ B = {1,2,3,4,5}
Intersection (A ∩ B)
Elements common to A and B.
Example: A = {1,2,3}, B = {3,4,5} → A ∩ B = {3}
Difference (A − B)
Elements in A not in B.
Example: A = {1,2,3}, B = {3,4,5} → A − B = {1,2}
Complement (A′)
All elements in universal set U not in A.
Example: U = {1,2,3,4,5}, A = {1,2} → A′ = {3,4,5}
