Graph Traversal

Graph traversal is the process of visiting every vertex in a graph in a systematic way. Traversing a graph helps us explore its structure and relationships between nodes. There are two main types of graph traversal: Breadth-First Search (BFS) and Depth-First Search (DFS).

Breadth-First Search (BFS)

BFS starts from a selected node and explores all its neighboring nodes first, before moving to the next level. It uses a queue to keep track of vertices to be visited.

Example: If the graph has edges (A–B), (A–C), (B–D), (C–E), then starting BFS from A gives → A, B, C, D, E.

Depth-First Search (DFS)

DFS explores as far as possible along a branch before backtracking. It uses a stack (or recursion). DFS is useful in detecting cycles, path finding, and topological sorting.

Example: Using the same graph, DFS starting from A gives → A, B, D, C, E.
