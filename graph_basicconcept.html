Graph Theory: Basic Concepts

Learn the fundamental components and ideas used in graph theory with examples and real-life applications.

A graph is a collection of vertices (nodes) connected by edges (links). Graphs are used to represent networks such as roads, communication lines, and social media connections.

Example:
G = (V, E) where
V = {A, B, C, D}
E = {(A, B), (A, C), (B, D)}

Real-Life Examples

  • Google Maps uses graphs to find the shortest route between two cities.
  • Social networks represent people as vertices and friendships as edges.
  • Computer networks use graphs to model connections between devices.

Representation of Graphs

Graphs can be represented in various ways, mainly using adjacency methods:

  • Adjacency Matrix: A 2D array where each cell (i, j) shows if an edge exists between vertices i and j.
  • Adjacency List: Each vertex has a list of adjacent vertices.

Common Graph Terminology

  • Degree: Number of edges incident to a vertex.
  • Path: A sequence of edges connecting vertices.
  • Cycle: A path where the start and end vertices are the same.
  • Connected Graph: Every vertex is reachable from any other vertex.
